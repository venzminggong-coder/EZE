# READYPH - Disaster Preparedness App

READYPH is an educational application designed to help users learn about disaster safety and hazard awareness in the Philippines through interactive modules, quizzes, and a hazard map.

## Features
*   **Learning Modules**: Comprehensive guides on Earthquakes, Typhoons, Volcanic Eruptions, and more.
*   **Interactive Hazard Map**: Explore regional risk levels and common hazards across the Philippines.
*   **AI-Powered Explanations**: Get real-time, concise explanations for quiz answers powered by the Google Gemini API.
*   **Gamified Progress**: Earn XP, level up, and maintain daily streaks as you master disaster preparedness.
*   **Cross-Platform**: Designed for a seamless experience on both mobile and desktop.

## Tech Stack
*   **Frontend**: React 19, Tailwind CSS, Lucide-React
*   **Build Tool**: Vite, TypeScript
*   **AI Integration**: Google GenAI (Gemini API)
*   **Data Sources**: PHIVOLCS, PAGASA, and NDRRMC

## Getting Started

### Prerequisites
*   A Google Gemini API Key

### Installation
1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open the application in your browser.

## Research Context
This application was originally developed as part of a study on interactive disaster education. It uses modern web technologies to provide a high-fidelity learning experience for students.
